# Hostwright Brand Assets

Bulky generated brand-source PNGs are not kept in the current public tree.

The private archive keeps the original exploratory PNGs for provenance review. They are not final production brand assets:

- They are RGB PNGs.
- They do not contain alpha transparency.
- They are not vector artwork.
- They should not be used as proof of final visual identity, trademark readiness, or production design system completeness.

Any future optimized images, SVGs, favicons, or social assets should be produced in a separate approved design pass.
