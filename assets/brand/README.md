# Hostwright Brand Source Assets

This folder contains copied source PNG assets for the Hostwright name and icon exploration.

These files are not final production brand assets:

- They are RGB PNGs.
- They do not contain alpha transparency.
- They are not vector artwork.
- They should not be used as proof of final visual identity, trademark readiness, or production design system completeness.

Use `assets/brand/originals/` for preserved raw inputs. Any future optimized images, SVGs, favicons, or social assets should be produced in a separate approved design pass.
